# 安裝方法

1. 找好要的資料夾位置，用`git clone https://github.com/Laxiflora/hexo_web_data`抓下來
2. 可以參考[本篇](https://laxiflora.github.io/2022/06/21/%E5%A6%82%E4%BD%95%E7%94%A8hexo%E5%9C%A8github%20page%E4%B8%8A%E9%83%A8%E7%BD%B2%E5%B1%AC%E6%96%BC%E8%87%AA%E5%B7%B1%E7%9A%84%E7%AC%AC%E4%B8%80%E5%80%8B%E7%B6%B2%E7%AB%99/)來引導安裝
3. 如果`npm install hexo-cli -g`失敗，可以嘗試把-g拿掉，換成`--save`

# 已使用的套件
- 為使網頁顯示完整，請安裝以下套件
```batch
npm install hexo-math --save
npm install hexo-admin --save
```
